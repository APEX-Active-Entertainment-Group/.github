# AAE GitHub Organization  
**Apex Active Entertainment Group – Internal Tools & Themes**

This is the official GitHub organization for **Apex Active Entertainment Group (AAE)**.

It is used to manage, version, and maintain **internal tools and Shopify themes** that support AAE’s vertically integrated active entertainment business.

---

## About AAE

**Apex Active Entertainment Group (AAE)** operates a portfolio of active entertainment businesses spanning manufacturing, facility operations, real estate, and digital systems.

AAE’s approach is defined by **Adventuretainment** — combining fun with physical challenge — and a focus on quality, durability, and disciplined execution.

More information: https://aae.group

---

## Purpose of This Organization

This GitHub organization exists to:

- Centralize development of AAE-owned digital tools and themes  
- Maintain clean version control and development history  
- Support long-term, maintainable internal systems  
- Enable AI-assisted development workflows (e.g. GitHub Copilot)  
- Provide a reliable technical record of how AAE builds and evolves its platforms  

All repositories in this organization are **private** and intended for internal use only.

---

## Repository Structure

### Internal Tools (WordPress)

The WordPress repositories in this organization are **internal tools** used across AAE properties for marketing, engagement, and operational support.

#### **WP-Daily-Deals**
Internal WordPress plugin for managing **rotating daily promotions** based on day-of-week logic.

- Use cases: recurring offers, automated promotions, campaign consistency  
- Technology: PHP / WordPress  

---

#### **WP-Popup-Game-System**
Internal WordPress tool for building **interactive, game-based popups** using HTML, CSS, and JavaScript.

Capabilities include:
- Visual theme controls  
- Prebuilt interaction templates  
- Asset management  
- Engagement and conversion tracking  
- Developer and no-code usage support  

- Technology: PHP, JavaScript, HTML, CSS  

---

### Shopify Themes

These repositories contain **Shopify themes** used by different spokes of AAE’s Shopify ecosystem. They are developed and maintained specifically for AAE-owned properties.

#### **AAI Theme**
Shopify theme and supporting assets for **Amusement Adventure Institute (AAI)**.

- Purpose: education, training, and content delivery  
- Technology: Shopify Liquid  

---

#### **Crazy Ape Theme**
Shopify theme for **Crazy Ape Extreme Equipment**, supporting both B2C and B2B use cases.

- Focus: performance, merchandising clarity, and brand alignment  
- Technology: Shopify Liquid
